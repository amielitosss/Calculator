# 🧮 Calculator

A simple, functional Calculator built with vanilla HTML, CSS, and JavaScript. Perform basic arithmetic (+, -, *, /), handle decimals, clear input, and enjoy smooth button animations. Calculation state persists via localStorage!

Calculator Demo

🧮 **Features**
- Number buttons 0-9, decimal (.), and operators +, -, *, /.
- Clear (C) button to reset input.
- Equals (=) evaluates expression (uses `eval()` for learning – not for production).
- Persistent calculation saved in browser's localStorage (survives refresh).
- Responsive design with dark gradient theme, hover effects, active scaling, and shadows.
- Mobile-friendly layout.
- No dependencies – runs instantly in any modern browser.

🚀 **How to Use**
- Click numbers and operators to build expression (e.g., 2 + 3 * 4).
- Hit **=** to compute result.
- Use **C** to clear and start over.
- Input persists on page reload!

📱 **How to Run**
- Open `index.html` in your web browser (double-click or drag to browser).
- Or in terminal: `start index.html` (Windows).
- Calculate immediately – no setup required!

**Demo display:**
```
2 + 3 * 4 = 14
```

🛠️ **Tech Stack**
| File          | Purpose                                      |
|---------------|----------------------------------------------|
| index.html    | UI structure, buttons with onclick handlers, result display |
| style.css     | Dark gradient theme, button styles (gray/orange/green), animations, shadows |
| script.js     | Logic: localStorage persistence, input append, DOM updates, `eval()` execution |

🔧 **Local Development**
- Edit files directly.
- Refresh browser to test changes.
- Calculation persists across sessions (clear browser data or use C to reset).
